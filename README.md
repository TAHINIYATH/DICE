# Dice Playing Website

A simple web application for playing a dice game with two players. The winner is determined by the player with the highest dice roll. In case of a tie, the game ends in a draw.


## Overview

This web application allows two players to roll a virtual dice, and the player with the higher dice number wins. If both players roll the same number, the game results in a draw. The project is implemented using HTML, CSS, and JavaScript to create a simple and interactive gaming experience.

## Features

- Two-player dice game
- Instant display of the winner or draw
- User-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. Open the [Dice Playing Website](#) in a web browser.
2. You will see two players, Player 1 and Player 2.
3. The player with the higher dice number wins.
4. If both players roll the same number, the game results in a draw.
